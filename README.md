## Install

```shell
$ go get -u https://gitlab.com/mikrowezel/backend/generator
$ alias mw=generator
```

Another option to avoid alias

```shell
$ git clone https://gitlab.com/mikrowezel/backend/generator.git
$ cd generator
$ make install
```
